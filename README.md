# Tracker
Tracker is a tool that helps find folders or files, anything thats a directory you just open it (I made a alias and named it tracker for simplicity) and it asks the name of the folder/file, you type it in and it finds all things with the name you just put in, then finds the first instance of it from the / directory and opens baobab to it, great for apps that leave stuff behind.
To clarify it this is to make it easier to find where files went by giving you direct access to the location, for example if something went wrong with a script or program and it moves files or folders to a unknown spot, or if you dont just want to go tot he system storage and search and instead want instant access to it. The program does have some bugs but from what I could test they do not affect the way it does its job and are merely just small issues in the code, feel free to fix them if you know how.
Also it has a custom feature, open the script and look at the overview and mainview section, overview is meant for the ones with ringview or treeview or whatever that shows you all paths and files, and mainview is meant for standard file managers like dolphin and nemo. If your device doesnt have the ones listed in quotations feel free to put in the name yourself for the ones you want it to use, great for if your system has multiple different apps and lets it be flexible.
If you don't have something like baobab and only have a main file manager such as nemo or dolphin just put both parts to the same app if you want to keep it simple.
